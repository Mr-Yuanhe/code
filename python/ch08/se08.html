8.8 网络编程

网络通信是两台计算机上的两个进程之间的通信。用Python进行网络编程，就是在Python程序本身这个进程内，连接别的服务器进程的通信端口进行通信。

Socket是网络编程的一个抽象概念。通常我们用一个Socket表示“打开了一个网络链接”，而打开一个Socket需要知道目标计算机的IP地址和端口号，再指定协议类型即可。

大多数连接都是可靠的TCP连接。创建TCP连接时，主动发起连接的叫客户端，被动响应连接的叫服务器。

下面以简单的数据收发为例，客户端输入数据，发送到服务端，服务器端生成（时间戳+数据）的封装数据回应客户端。由于socket编程包括两种：面向连接的和无连接的，这两种分别对应TCP数据流和UDP数据报文。所以，我们两种方法都进行实现。

面向连接的TCP socket编程

服务器端：\8.7-TCPSocket\8.7-server.py

from socket import *
from time import ctime

# 服务器的地址和端口
HOST = ''
PORT = 21567
ADDR = (HOST, PORT)

# 缓冲区大小为1K
BUFSIZ = 1024

# 创建服务器socket
tcpSerSock = socket(AF_INET, SOCK_STREAM)
# 绑定地址socket
tcpSerSock.bind(ADDR)
# 最大监听 5 client
tcpSerSock.listen(5)

try:
    while True:
        print('waiting for connection...')
        # build client socket
        tcpCliSock, addr = tcpSerSock.accept()
        print('...connect from:', addr)

        # accept data and process
        while True:
            data = tcpCliSock.recv(BUFSIZ).decode()
            print(data)
            if not data:
                break
            tcpCliSock.send(('[%s] %s' % (ctime(), data)).encode())

        # close client socket
        tcpCliSock.close()
except (EOFError, KeyboardInterrupt):
    tcpSerSock.close()

客户端：\8.7-TCPSocket\8.7-client.py

#!/usr/bin/python3
from socket import *

# Address and Port
HOST = '192.168.31.165'
PORT = 21567
ADDR = (HOST, PORT)

# BufferSize
BUFSIZ = 1024

#build socket
tcpCliSocket = socket(AF_INET, SOCK_STREAM)
tcpCliSocket.connect(ADDR)

while True:
    data = input('> ')
    if not data:
        break
    # send data
    tcpCliSocket.send(data.encode())
    # recv data
    data = tcpCliSocket.recv(BUFSIZ).decode()
    # show data
    print(data)
tcpCliSocket.close()

bye

waiting for connection...

无连接的UDP socket编程

服务器端：\8.8-UDPSocket\8.8-server.py

from socket import *
from time import ctime

# Address and Port
HOST = ''
PORT = 8000
ADDR = (HOST, PORT)

# BufferSize
BUFFSIZE = 1024
# build socket
udpSerSock = socket(AF_INET, SOCK_DGRAM)
# bind socket
udpSerSock.bind(ADDR)

try:
    while True:
        print('waiting the message...')
        data, addr = udpSerSock.recvfrom(BUFFSIZE)
        data = data.decode()
        print('received the message: '+data+' from: ', addr)
        udpSerSock.sendto(('[%s] %s' % (ctime(), data)).encode(), addr)
except (EOFError, KeyboardInterrupt):
    udpSerSock.close()

客户端：\8.8-UDPSocket\8.8-client.py

#!/usr/bin/python3
from socket import *

# Address and Port
HOST = '192.168.31.165'
PORT = 8000
ADDR = (HOST, PORT)

# BufferSize
BUFSIZ = 1024

# build socket
udpCliSock = socket(AF_INET, SOCK_DGRAM)

while True:
    data = input('> ')
    if not data:
        break
    udpCliSock.sendto(data.encode(), ADDR)
    data = udpCliSock.recvfrom(BUFSIZ)
    print(data)
udpCliSock.close()
