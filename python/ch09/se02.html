9.2 基本概念

类与对象是面向对象编程的两个主要方面。一个类（Class）能够创建一种新的类型（Type），其中对象（Object）就是类的实例（Instance）。

拥有相同(或者类似)属性和行为的对象都可以抽像出一个类，类就是一个模板，类定义了对象都有的属性和行为。

对象可以使用属于它的普通变量来存储数据。这种从属于对象或类的变量叫作字段（Field）。对象还可以使用属于类的函数来实现某些功能，这种函数叫作类的方法（Method）。这两个术语很重要，它有助于我们区分函数与变量，哪些是独立的，哪些又是属于类或对象的。总之，字段与方法通称类的属性（Attribute）。但通常也用属性指数据部分，方法指函数。

上一节的例子，在Python中使用class关键字定义类，使用class关键字与类的名称来创建一个新类。在它之后是一个缩进的语句块，代表这个类的主体。

# 创建类使用class关键字

# 创建类，类名一般遵循大驼峰命名方法。
class Student:
    pass

# 创建对象
s = Student()
print(s)
print(type(s))

定义类之后，用类名()的格式创建对象，s的类型为Student类，执行结果为

<__main__.Student object at 0x0000020D572BB208>
<class '__main__.Student'>

字段（属性）

有了对象之后，就可以给对象添加属性，并使用：

# 给对象添加属性
s.name = 'zhangsan'
print(s.name)

方法

在类的定义中，给类添加方法，类的方法至少有一个self参数：

    # 定义方法
    def baoming(self):
        print('学生报名。')

使用对象调用方法：

# 调用方法
s.baoming()

方法中可以使用对象的属性，前提是这个属性存在：

    def baoming(self):
        print('学生{}报名。'.format(self.name))

self必须添加到参数列表的开头，但是你不用在你调用这个功能时为这个参数赋值，Python会为它提供。self变量引用的是对象本身，按照惯例，它被赋予self这一名称。

本节代码：9.1-创建类.py

# 创建类使用class关键字

# 创建类，类名一般遵循大驼峰命名方法。
class Student:
    # 定义方法
    def baoming(self):
        print('学生{}报名。'.format(self.name))


# 创建对象
s = Student()
# 给对象添加属性
s.name = 'zhangsan'
print(s.name)
# print(s)
# print(type(s))

# 调用方法
s.baoming()
