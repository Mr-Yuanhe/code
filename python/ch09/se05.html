9.5 类变量和类方法

字段（Field）有两种类型——类变量与对象变量，它们根据究竟是类还是对象拥有这些变量来进行分类。

之前的对象变量（Object variable）由类的每一个独立的对象或实例所拥有。在这种情况下，每个对象都拥有属于它自己的字段的副本，也就是说，它们不会被共享，也不会以任何方式与其 它不同实例中的相同名称的字段产生关联。

而类变量（Class Variable）是共享的（Shared），即它们可以被属于该类的所有实例访问。 该类变量只拥有一个副本，当任何一个对象对类变量作出改变时，发生的变动将在其它所有 实例中都会得到体现。

定义一个类变量是在类的内部，所有类函数的外部。

类方法是一个属于类而非属于对象的方法。一般用来对类变量进行操作。在Python中使用装饰器（Decorator）@classmethod将一个方法标记为类方法。

相对于self是代表某一个对象本身，cls代表这个类本身，可以用cls来引用类变量。

例如给学生类加入一个类变量和类方法：9.4-类变量.py

# 类变量

# 创建类，类名一般遵循大驼峰命名方法。
class Student:
    '''
        Student类说明文档
    '''

    # 类变量，标明学生数目
    stuNumber = 0

    # 带其他参数的构造函数
    def __init__(self, name, sex, age):
        self.name = name
        self.sex = sex
        self.age = age
        Student.stuNumber += 1

    def __str__(self):
        return '学生：{}，性别：{}，年龄：{}。'.format(self.name, self.sex, self.age)

    # 定义方法
    def baoming(self):
        print('学生{}报名。'.format(self.name))

    def biye(self):
        print('学生{}毕业了。'.format(self.name))
        Student.stuNumber -= 1

    @classmethod
    def how_many(cls):
        print('学生数目：{}'.format(cls.stuNumber))


# 带参数的构造函数，创建对象
s1 = Student('lisi', 'male', 24)
s2 = Student('wangwu', 'male', 23)
Student.how_many()
s1.biye()
Student.how_many()
s2.biye()
Student.how_many()

执行结果为：

学生数目：2
学生lisi毕业了。
学生数目：1
学生wangwu毕业了。
学生数目：0
