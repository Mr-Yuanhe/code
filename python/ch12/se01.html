1. 猜数字游戏

（第5章结束项目）

编写一个程序，进行玩猜数字的游戏。一个没有重复数字的4位数，猜对数字和位置显示为A，数字对，位置不对为B。例如数字为1234，猜3564，则显示1A1B，一共10轮猜出数字为胜利。

使用random模块，使用方法：

import random

# 生成0-9的随机数字
n = random.randint(0, 9)

效果：

生成数字。。。
第1次，请输入猜的数字：1234
结果为：0A1B
第2次，请输入猜的数字：5678
结果为：0A1B
第3次，请输入猜的数字：1290
结果为：0A2B
第4次，请输入猜的数字：9034
结果为：0A3B
第5次，请输入猜的数字：0935
结果为：0A3B
第6次，请输入猜的数字：3609
结果为：2A0B
第7次，请输入猜的数字：5409
结果为：2A2B
第8次，请输入猜的数字：4509
结果为：4A0B
正确！
